Health Data Dashboard

Problem Statement/Purpose

To address the challenge of real-time health data monitoring in underserved regions, this project aims to provide an accessible visualization tool for health officials to make informed decisions quickly.

Technology Used

Power BI, Python, SurveyCTO API, Excel

Description

Developed a dynamic dashboard that integrates survey data with administrative records, enabling instant tracking of key health metrics like vaccination rates, disease prevalence, and hospital bed occupancy. The tool supports filtering by region and time period, with automated alerts for anomalies, and has been adopted by local health departments for weekly reviews.

Survey Automation Tool

Problem Statement/Purpose

To reduce manual effort in processing large survey datasets, this tool aims to automate repetitive cleaning tasks, improving efficiency for research teams.

Technology Used

Stata, Python, R

Description

Created custom Stata scripts to automate data recoding, validation, and outlier detection, significantly cutting down processing time for multi-variable surveys. The tool handles multi-select variables, generates quality reports, and includes a Python-based scheduler, improving accuracy by 15% and reducing workload by 30% for a team of 10 analysts.

Child Nutrition Study

Problem Statement/Purpose

To identify key factors contributing to child malnutrition in rural areas, aiming to inform targeted interventions and policy recommendations.

Technology Used

Stata, R, SurveyCTO, GIS

Description

Conducted a field study analyzing nutritional data from 500 households across 10 rural districts, producing actionable insights for health policy development. Key findings included socioeconomic correlations with stunting rates and dietary patterns, leading to recommendations for local programs and a GIS-based mapping of affected zones.
